DLTP JavaScript Library Copies

This page links to copies of JavaScript files from the Duke Learn To Program (DLTP) Library. The intent is to allow hosting of these JavaScript files for use by web pages requiring serving over HTTPS. The DLTP library is not served over HTTPS, so they can't be used on services such as Netlify. I intend to host these copies on Netlify.

DLTP makes the source code available via the Programming Foundations with JavaScript, HTML and CSS course.

File Name Description URL
SimpleImage.js Used to create, manipulate, and output images. DLTP Documentation https://simpleimagejs.netlify.app/simpleimage.js
SimplePixel.js Used to access and manipulate pixels of SimpleImage objects. DLTP Documentation https://simpleimagejs.netlify.app/simpleimage.js
